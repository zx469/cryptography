# 实验分析

## 一、使用的分析方法
**多因素重复使用密钥流攻击（Many-time pad attack）**  
  原理：当多个密文使用相同密钥流时，对每个字节位置，遍历所有可能的密钥字节（0~255），若该密钥能使所有密文在该位置解密出的字符均为可打印ASCII（32~126），则将该密钥字节作为密钥流对应位置的值。  
  实现：通过暴力枚举每个位置的密钥字节，结合明文可打印字符的约束条件，逐步还原完整密钥流，最后用密钥流解密目标密文。

## 二、如何确认目标密文的明文内容
1. **密钥流恢复**：对每个字节位置，尝试所有候选密钥，仅保留使所有密文解密结果均为可打印字符的密钥字节。  
2. **解密目标密文**：将恢复的密钥流与目标密文逐字节异或，得到解密后的字节序列。  
3. **内容验证**：解密得到的字节序列全部落在可打印ASCII范围内，且形成的英文句子具有完整语义，从而确认攻击有效。

## 三、解密得到的明文
The secret message is: when using a stream cipher, never use the key more than once.
