# Lab2 多次填充攻击流密码 实验报告
## 1 攻击方法
1. 流密码加密模型：$C = M \oplus K$，同一密钥K加密所有密文
2. 两密文异或：$C_a \oplus C_b = M_a \oplus M_b$，彻底消除密钥流
3. 利用特征：空格`0x20`与英文字母异或会翻转大小写；通过多组明文异或交叉比对，定位所有空格位置
4. 逐步还原多条已知明文，用 $K=C_i\oplus M_i$ 反推密钥流，多组校验保证密钥流唯一正确
5. 最终用密钥流异或目标密文，得到目标明文

## 2 密钥流推断过程
1. 先对11组密文两两做字节异或，得到大量 $M_i\oplus M_j$ 数据
2. 观察异或结果中大小写翻转特征，标记每个位置是否存在空格
3. 结合英文常用句式、单词长度，拼接还原多条完整明文句子
4. 用任意一组「密文+还原明文」计算密钥流，再用其他密文+明文验证密钥流一致性，剔除错误推断
5. 收敛得到全局唯一密钥流

## 3 最终解密结果
目标密文解密后的明文为可读英文句子（程序运行直接输出ASCII明文）
