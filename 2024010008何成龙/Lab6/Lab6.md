# Lab6：基于哈希链的文件认证系统
## 第1题 计算 intro.mp4 的哈希链根哈希值
intro.mp4 的哈希链根哈希值 h0：a89de1b01b244f5b29b1edb5f70365b2f8c8c7d7b1abc7c17145135586c830fc

## 实现说明
1. 开发语言：Python
2. 依赖库：pycryptodome
3. 核心步骤：
   - 二进制读取文件，转换为十六进制字符串
   - 按2048个字符（1KB）分块并反转
   - 从最后一块开始迭代拼接哈希，计算SHA256
   - 最终得到根哈希h0
4. 验证：test.mp4计算结果与官方标准值完全一致
