一、分析方法

采用流密码密钥重用攻击，利用同一密钥流加密多组明文的漏洞：

1. 加密关系：C_i = P_i \oplus K
2. 两组密文异或可消去密钥：C_i \oplus C_j = P_i \oplus P_j
3. 利用英文空格与字母异或仍为字母的特征，推断空格位置并还原密钥流，最终解密目标密文。

二、明文确认方法

通过密文异或统计高频字母位置，推测空格并推导密钥；
用还原密钥解密其他密文，验证是否为可读英文文本，确认明文正确。

三、解密结果

目标密文明文：
The secret message is: Never use stream cipher key more than once.
四、实验结论

流密码密钥重复使用会导致严重安全漏洞，攻击者可通过多组密文快速还原明文。
