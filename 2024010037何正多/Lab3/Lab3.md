## 第 20 题（问答/证明题，3 分）

**PRG 优势（Advantage）的定义与计算**

设 $G: K \to \{0,1\}^n$ 是一个 PRG，$A$ 是 $\{0,1\}^n$ 上的统计测试。

**(a)** 请写出 PRG 优势 $\mathrm{Adv}_{\mathrm{PRG}}[A, G]$ 的定义式。

**答案：**

```


答：

PRG 优势用于量化统计测试 A 区分伪随机序列与真随机序列的能力，定义式为：

$$
\text{Adv}_{\text{PRG}}[A, G] = \left| \Pr_{k \leftarrow K}[A(G(k)) = 1] - \Pr_{x \leftarrow \{0,1\}^n}[A(x) = 1] \right|
$$

---

各部分解析：

- $ \Pr_{k \leftarrow K}[A(G(k)) = 1] $：从密钥空间 K 均匀随机选取密钥 k，A 输出 1 的概率（伪随机序列场景）

- $ \Pr_{x \leftarrow \{0,1\}^n}[A(x) = 1] $：从真随机序列空间 $ \{0,1\}^n $ 均匀随机选取 x，A 输出 1 的概率（真随机序列场景）

- 取绝对值是为了消除概率差的正负影响，仅衡量区分能力的强弱

```

**(b)** 解释 $\mathrm{Adv}_{\mathrm{PRG}}[A, G]$ 接近 1 和接近 0 分别意味着什么。

**答案：**

```
**Adv_PRG[A, G]** 是衡量攻击者 A 能否有效区分伪随机生成器 G 的输出与真随机字符串的优势值，其定义为：

> **Adv_PRG[A, G] = | Pr[A(G(s)) = 1] - Pr[A(r) = 1] |**

其中：
- s ← {0,1}^n：均匀随机选取种子；
- r ← {0,1}^ℓ：均匀随机选取长度为 ℓ 的真随机串（ℓ > n）；
- A 是一个概率多项式时间（PPT）算法（即攻击者/区分器）。

---

 当 Adv_PRG[A, G] 接近 1 时：

表示攻击者 A **极有可能正确区分** G(s) 与真随机串 r。  
→ 说明伪随机生成器 G 的输出**不具备不可区分性**，即“不够随机”，容易被识别。  
→ **PRG 不安全**，不能用于密码学安全场景（如密钥生成、流密码等）。

**通俗理解**：攻击者几乎总能猜出“这是不是 PRG 生成的”，安全防线崩溃。

---

当 Adv_PRG[A, G] 接近 0 时：

表示攻击者 A **几乎无法区分** G(s) 与真随机串 r。  
→ 说明 G 的输出在计算上与真随机无异，对多项式时间攻击者“不可分辨”。  
→ **PRG 安全**，满足密码学中对伪随机性的基本要求。

**通俗理解**：攻击者猜对的概率≈50%，等同于瞎猜，安全有效。

---
```

**(c)** 若 $A(x) = 0$（即 $A$ 对任何输入都输出 0），求 $\mathrm{Adv}_{\mathrm{PRG}}[A, G]$ 的值。

**答案：**

```
已知：  
**A(x) ≡ 0**，即对于任意输入 x，攻击者 A 都输出 0。

代入优势定义：

> **Adv_PRG[A, G] = | Pr[A(G(s)) = 1] - Pr[A(r) = 1] |**

由于 A 恒输出 0：

→ Pr[A(G(s)) = 1] = 0  
→ Pr[A(r) = 1] = 0  

因此：

> **Adv_PRG[A, G] = |0 - 0| = 0**

---

 **最终答案**：  
**Adv_PRG[A, G] = 0**

---

```

## 第 21 题（问答/证明题，4 分）

**PRG 优势的具体计算**

假设 $G: K \to \{0,1\}^n$ 满足：对于 $K$ 中 $\frac{2}{3}$ 的密钥 $k$，有 $\mathrm{MSB}(G(k)) = 1$（即输出的最高有效位为 1）。

定义统计测试 $A(x)$ 如下：
$$A(x) = \begin{cases} 1 & \text{若 } \mathrm{MSB}(x) = 1 \\ 0 & \text{否则} \end{cases}$$

**(a)** 计算 $\Pr[A(G(k)) = 1]$，其中 $k \xleftarrow{R} K$。

**答案：**

```
求解目的：  
计算当输入为 PRG 的输出 $ G(k) $ 时，测试器 $ A $ 输出 1 的概率 —— 即 PRG 输出最高位为 1 的概率。

解答过程：  
根据题设，“对于 $ K $ 中 $ \frac{2}{3} $ 的密钥 $ k $，有 $ \text{MSB}(G(k)) = 1 $”，  
而 $ k $ 是从 $ K $ 中**均匀随机选取**的（记作 $ k \stackrel{R}{\longleftarrow} K $），  
所以：

$$
\operatorname{Pr}[A(G(k)) = 1] = \operatorname{Pr}[\text{MSB}(G(k)) = 1] = \frac{2}{3}
$$

---

```

**(b)** 计算 $\Pr[A(r) = 1]$，其中 $r \xleftarrow{R} \{0,1\}^n$。

**答案：**

```
求解目的：  
计算当输入为真正的随机串 $ r \in \{0,1\}^n $ 时，测试器 $ A $ 输出 1 的概率 —— 即随机串最高位为1的概率。

解答过程：  
由于 $ r $ 是从 $ \{0,1\}^n $ 中**均匀随机选取**的，  
那么其最高有效位（MSB）为 0 或 1 的概率各占一半：

$$
\operatorname{Pr}[A(r) = 1] = \operatorname{Pr}[\text{MSB}(r) = 1] = \frac{1}{2}
$$

```

**(c)** 计算 $\mathrm{Adv}_{\mathrm{PRG}}[A, G]$。

**答案：**

```
求解目的：  
计算攻击者 $ A $ 对 PRG $ G $ 的**优势**（advantage），这是衡量 PRG 安全性的核心指标。

公式回顾：  
PRG 的优势定义为：

$$
\operatorname{Adv}_{PRG}[A, G] = \left| \operatorname{Pr}[A(G(k)) = 1] - \operatorname{Pr}[A(r) = 1] \right|
$$

代入计算：

$$
\operatorname{Adv}_{PRG}[A, G] = \left| \frac{2}{3} - \frac{1}{2} \right| = \left| \frac{4 - 3}{6} \right| = \frac{1}{6}
$$

--
```

**(d)** 该 PRG 是否安全？为什么？

**答案：**

```
判断标准：  
一个 PRG 被认为是**安全的**，当且仅当对任何高效统计测试器 $ A $，其优势 $ \operatorname{Adv}_{PRG}[A, G] $ 是**可忽略的**（negligible），即随着安全参数增长趋近于0。

但在本题中，我们找到了一个具体的统计测试器 $ A $，其优势为：

$$
\operatorname{Adv}_{PRG}[A, G] = \frac{1}{6}
$$

这是一个**常数级**的优势（不随 $ n $ 增大而变小），远非“可忽略”。
**该 PRG 不安全**。  
> 因为存在一个高效统计测试器 $ A $（仅检查最高位），能以显著优势（$ \frac{1}{6} $）区分 PRG 的输出与真随机串 —— 违反了伪随机生成器“不可区分性”的基本安全要求。

```

## 第 22 题（证明题，5 分）

**证明 OTP 具有完美保密性**

**引理**：一次性密码本（OTP）具有完美保密性（Perfect Secrecy）。

**(a)** 写出完美保密性的定义：对于任意明文 $m$ 和密文 $c$，$\Pr_k[E(k,m) = c]$ 应满足什么条件？

**答案：**

```

---

## (a) 完美保密性的定义

**求解目的**：明确“完美保密性”的数学定义，为后续证明建立理论基础。

> 对于任意明文 $ m \in \mathcal{M} $ 和密文 $ c \in \mathcal{C} $，  
> 完美保密性要求：  
> **加密后得到密文 $ c $ 的概率与明文 $ m $ 无关**，即：  
> $$
\Pr_k[E(k, m) = c] = \Pr_k[E(k, m') = c] \quad \text{对任意 } m, m' \in \mathcal{M}
$$  
> 或等价地，**对任意固定的 $ c $，该概率为常数（不依赖于 $ m $）**。  
> 更严格地，可表述为：  
> $$
\Pr_k[E(k, m) = c] = \Pr[c] \quad \text{且独立于 } m
$$  
> 也就是说，在不知道密钥的前提下，观察到密文 $ c $ 不会泄露任何关于明文 $ m $ 的信息。

**标准定义（Shannon）**：  
一个加密方案满足完美保密性，当且仅当对所有 $ m \in \mathcal{M}, c \in \mathcal{C} $，有：  
$$
\Pr[M = m \mid C = c] = \Pr[M = m]
$$  
即，**给定密文 $ c $，明文 $ m $ 的后验概率等于其先验概率** —— 密文不提供任何额外信息。

从加密函数角度，等价条件为：  
$$
\forall m \in \mathcal{M}, \forall c \in \mathcal{C}, \quad \Pr_k[E(k, m) = c] \text{ 是常数（独立于 } m \text{）}
$$

---
```

**(b)** 对于 OTP，证明对任意 $m \in \mathcal{M}$ 和 $c \in \mathcal{C}$，恰好存在多少个密钥 $k$ 使得 $E(k, m) = c$？

**答案：**

```
**求解目的**：分析 OTP 的加密函数结构，确定满足 $ E(k, m) = c $ 的密钥数量，为 (c) 的概率计算做铺垫。

> 在 OTP 中：  
> - 明文空间 $ \mathcal{M} = \{0, 1\}^n $  
> - 密钥空间 $ \mathcal{K} = \{0, 1\}^n $（与明文等长，且均匀随机）  
> - 密文空间 $ \mathcal{C} = \{0, 1\}^n $  
> - 加密函数：$ E(k, m) = k \oplus m $（逐位异或）  
> - 解密函数：$ D(k, c) = k \oplus c $

**现在，固定 $ m \in \mathcal{M}, c \in \mathcal{C} $，求满足 $ E(k, m) = c $ 的 $ k $ 的个数。**

由 $ E(k, m) = k \oplus m = c $，  
得：$ k = c \oplus m $

→ **唯一解！**

即：对于任意给定的 $ m $ 和 $ c $，**有且仅存在一个密钥 $ k = c \oplus m $** 使得加密结果为 $ c $。

**结论**：  
> 对任意 $ m \in \mathcal{M}, c \in \mathcal{C} $，**恰好存在 1 个密钥 $ k $** 使得 $ E(k, m) = c $。

**(c)** 利用 (a) 和 (b) 的结论，完成 OTP 具有完美保密性的证明。

由 (b)：对任意 $ m, c $，存在**唯一**密钥 $ k = c \oplus m $ 使得 $ E(k, m) = c $。

\Pr[k] = \frac{1}{|\mathcal{K}|} = \frac{1}{2^n}
$$

\Pr_k[E(k, m) = c] = \Pr[k = c \oplus m] = \frac{1}{2^n}

也就是说：

## 第 23 题（证明题，5 分）

**语义安全性与明文信息泄露**

假设存在一个高效算法 $A$，它总是能从密文 $c$ 中推断出明文 $m$ 的最低有效位 $\mathrm{LSB}(m)$。

**(a)** 请证明：加密方案 $\mathbb{E} = (E, D)$ 不是语义安全的。

**答案：**

```
**语义安全（Semantic Security）** 的核心定义是：在“选择明文攻击”（CPA）下，攻击者无法区分两个等长明文加密后的密文。更形式化地说，在语义安全游戏中，攻击者的优势应趋近于 0。

现假设存在高效算法 $ A $，对任意密文 $ c = E(k, m) $，总能输出 $ \text{LSB}(m) $。

我们可构造如下攻击策略：

- 攻击者选择两个明文 $ m_0, m_1 $，满足 $ \text{LSB}(m_0) \neq \text{LSB}(m_1) $（例如 $ m_0 = 0 $, $ m_1 = 1 $）。
- 挑战者随机选择 $ b \in \{0,1\} $，加密 $ m_b $ 得到密文 $ c $。
- 攻击者运行算法 $ A(c) $，得到 $ \text{LSB}(m_b) $。
- 若输出为 $ \text{LSB}(m_0) $，则猜 $ b=0 $；否则猜 $ b=1 $。

由于 A 总是正确输出 LSB，攻击者总能猜对 $ b $，即优势为 1。

→ **这违反了语义安全的定义（优势应 ≤ negligible）**。  
**因此，加密方案 $ \mathbb{E} $ 不是语义安全的。**

---

```

**(b)** 构造一个攻击者 $B$，使用 $A$ 作为子程序来赢得语义安全游戏。具体地：
   - 攻击者 $B$ 应选择哪两条消息 $m_0, m_1$ 发送给挑战者？
   - 收到密文 $c$ 后，$B$ 如何利用 $A$ 来猜测 $b$？

**答案：**

```
#### 1. 攻击者 B 应选择哪两条消息 $ m_0, m_1 $ 发送给挑战者？

> **选择两个明文，其 LSB 不同。**

最简单构造：  
- $ m_0 = 0 $（或任意偶数） → $ \text{LSB}(m_0) = 0 $  
- $ m_1 = 1 $（或任意奇数） → $ \text{LSB}(m_1) = 1 $

（只要满足 $ \text{LSB}(m_0) \neq \text{LSB}(m_1) $ 即可，如 $ m_0 = 2, m_1 = 3 $ 等亦可）

---

#### 2. 收到密文 $ c $ 后，B 如何利用 A 来猜测 b？

> **调用算法 A，根据其输出的 LSB 值反推 b。**

具体步骤：

1. B 向挑战者提交 $ (m_0, m_1) $，其中 $ \text{LSB}(m_0) = 0, \text{LSB}(m_1) = 1 $。
2. 挑战者随机选 $ b \in \{0,1\} $，计算 $ c = E(k, m_b) $ 并返回给 B。
3. B 调用子程序 $ A(c) $，得到 $ \hat{b} = \text{LSB}(m_b) $。
4. 若 $ \hat{b} = 0 $，则输出猜测 $ b' = 0 $；  
   若 $ \hat{b} = 1 $，则输出猜测 $ b' = 1 $。
5. 返回 $ b' $ 作为最终猜测。

由于 A 总是正确输出 LSB，故 B 总能正确猜出 $ b $。

---

```

**(c)** 计算 $\mathrm{Adv}_{\mathrm{SS}}[B, \mathbb{E}]$ 的值。

**答案：**

```
语义安全优势定义为：

$$
\text{Adv}_{\text{SS}}[B, \mathbb{E}] = \left| \Pr[b' = b] - \frac{1}{2} \right|
$$

其中：

- $ \Pr[b' = b] $ 是攻击者猜对的概率。
- 由于 B 总是能通过 A 正确获得 LSB，进而 100% 猜对 $ b $，所以：

$$
\Pr[b' = b] = 1
$$

→ 因此：

$$
\text{Adv}_{\text{SS}}[B, \mathbb{E}] = \left| 1 - \frac{1}{2} \right| = \frac{1}{2}
$$

## 第 24 题（证明题，5 分）

**证明 OTP 是语义安全的**

考虑语义安全游戏的两个实验：
- **EXP(0)**：挑战者加密 $m_0$，即 $c \leftarrow k \oplus m_0$
- **EXP(1)**：挑战者加密 $m_1$，即 $c \leftarrow k \oplus m_1$

其中 $k \xleftarrow{R} K$，$|m_0| = |m_1|$。

**(a)** 证明在 EXP(0) 中，密文 $c = k \oplus m_0$ 在 $\{0,1\}^n$ 上是均匀分布的。

**答案：**

```
**答案：**
要证明 $\text{EXP}(0)$ 中密文 $c = k \oplus m_0$ 在 $\{0,1\}^n$ 上均匀分布，推导如下：

1.  **密钥均匀性**：密钥 $k$ 从密钥空间 $K = \{0,1\}^n$ 中均匀随机选取，故对任意 $k' \in \{0,1\}^n$，有
    $$
    \Pr[k = k'] = \frac{1}{2^n}
    $$

2.  **异或映射双射性**：对固定的 $m_0$，映射 $f: k \mapsto k \oplus m_0$ 是双射。
    *   单射性：若 $k_1 \oplus m_0 = k_2 \oplus m_0$，两边异或 $m_0$ 得 $k_1 = k_2$。
    *   满射性：对任意 $c' \in \{0,1\}^n$，取 $k' = c' \oplus m_0$，则 $f(k') = c'$。

3.  **密文均匀性推导**：对任意 $c' \in \{0,1\}^n$，
    $$
    \Pr[c = c'] = \Pr[k \oplus m_0 = c'] = \Pr[k = c' \oplus m_0] = \frac{1}{2^n}
    $$
    因此 $c$ 在 $\{0,1\}^n$ 上均匀分布。
```

**(b)** 证明在 EXP(1) 中，密文 $c = k \oplus m_1$ 在 $\{0,1\}^n$ 上也是均匀分布的。

**答案：**

```
**答案：**
证明逻辑与 (a) 完全一致，仅将 $m_0$ 替换为 $m_1$：

1.  密钥 $k$ 仍从 $\{0,1\}^n$ 中均匀随机选取，满足 $\Pr[k = k'] = \frac{1}{2^n}$。
2.  对固定的 $m_1$，映射 $g: k \mapsto k \oplus m_1$ 同样是 $\{0,1\}^n$ 到自身的双射。
3.  对任意 $c' \in \{0,1\}^n$，
    $$
    \Pr[c = c'] = \Pr[k \oplus m_1 = c'] = \Pr[k = c' \oplus m_1] = \frac{1}{2^n}
    $$
    因此 $c = k \oplus m_1$ 在 $\{0,1\}^n$ 上均匀分布。
```

**(c)** 由 (a) 和 (b) 说明 EXP(0) 和 EXP(1) 中密文的分布是相同的（identical distributions）。

**答案：**

```
**答案：**
由 (a) 可知，$\text{EXP}(0)$ 中密文 $c_0 = k \oplus m_0$ 的分布为：
$$
\forall c' \in \{0,1\}^n, \Pr[c_0 = c'] = \frac{1}{2^n}
$$

由 (b) 可知，$\text{EXP}(1)$ 中密文 $c_1 = k \oplus m_1$ 的分布为：
$$
\forall c' \in \{0,1\}^n, \Pr[c_1 = c'] = \frac{1}{2^n}
$$

由于两个实验中密文的概率质量函数完全相同，因此 $\text{EXP}(0)$ 和 $\text{EXP}(1)$ 中的密文分布** identical**（完全一致），即 $c_0 \stackrel{d}{=} c_1$。
```

**(d)** 利用 (c) 的结论，证明对于任意攻击者 $A$：
$$\mathrm{Adv}_{\mathrm{SS}}[A, \mathrm{OTP}] = \left| \Pr[A(k \oplus m_0) = 1] - \Pr[A(k \oplus m_1) = 1] \right| = 0$$

**答案：**

```
**答案：**
语义安全优势定义为：
$$
\text{Adv}_{\text{SS}}[A, \text{OTP}] = \left| \Pr[A(c_0) = 1] - \Pr[A(c_1) = 1] \right|
$$

由 (c) 的结论知 $c_0$ 与 $c_1$ 分布完全相同，根据概率分布的性质，对于任意攻击者 $A$，随机变量 $A(c_0)$ 与 $A(c_1)$ 的分布也完全相同，故：
$$
\Pr[A(c_0) = 1] = \Pr[A(c_1) = 1]
$$

将该等式代入优势定义式，可得：
$$
\text{Adv}_{\text{SS}}[A, \text{OTP}] = \left| p - p \right| = 0
$$
其中 $p = \Pr[A(c_0) = 1] = \Pr[A(c_1) = 1]$。

因此，OTP 方案满足语义安全。
```
