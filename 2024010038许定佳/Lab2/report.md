## 一、实验目的
利用流密码重复使用密钥的安全漏洞，实现多次填充攻击，破解目标密文。

## 二、实验原理
1. 流密码加密：C = M ⊕ 密钥流
2. 相同密钥流加密：C1 ⊕ C2 = M1 ⊕ M2
3. 空格(0x20)与字母异或会发生大小写互换
4. 统计所有密文异或结果，推断空格位置 → 计算密钥流 → 解密密文

## 三、攻击方法
1. 将所有密文从十六进制转为字节数组
2. 遍历每个字节位置，统计所有密文对的异或值
3. 对符合“空格+字母”特征的密钥候选值进行打分
4. 取每个位置最高分作为密钥字节
5. 使用密钥流解密目标密文得到明文

## 四、实验结果
### 解密得到的明文：
The secret message is: When using a stream cipher, never use the same key more than once!

## 五、实验结论
流密码严禁重复使用相同密钥，否则攻击者可通过密文异或消除密钥流，结合字符特征实现完全破解。
